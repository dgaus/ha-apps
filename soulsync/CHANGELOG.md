# Changelog

## 2.7.2

- Initial release, wrapping upstream [SoulSync 2.7.2](https://github.com/Nezreka/SoulSync).
- Web UI on port 8008; Spotify/Tidal OAuth callback ports 8888/8889 (optional).
- Library and downloads under `/media` (pairs with the slskd app).
- Persistent config, database and logs in `/data/soulsync`.
- Logs the sibling slskd app's internal address for easy setup.
